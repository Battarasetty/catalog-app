# Dynamic Multi-Category Product Catalog

A responsive product catalog built as part of a frontend developer assignment.

Live Demo :-
https://catalog-app-pbtw.vercel.app

What it does :-
- Displays products across 4 categories: Cars, Bikes, Phones, Computers
- Each category section shows item cards with a preview of specs
- Clicking any item opens a detail page that dynamically renders
  all specifications from itemprops[] — same component works for every category
- Back button returns to the catalog

## Tech Stack
- Next.js 14 (App Router)
- TypeScript
- Material UI v9

## Run Locally
npm install
npm run dev

Open http://localhost:3000
